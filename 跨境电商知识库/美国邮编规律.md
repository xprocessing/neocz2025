美国邮政编码（ZIP Code）是5位数字（有时后面加4位扩展码，写成ZIP+4，如12345-6789），其前几位数字有非常明显的地理规律，从东到西、从北到南大致递增。下面是美国ZIP Code的主要规律总结：

### 1. 第一位数字（区号，Zone）：代表美国的大区域
| 第一位数字 | 主要覆盖地区（从东北向西、向南递增） |
|------------|-------------------------------------|
| 0          | 东北部：新英格兰（康涅狄格CT、缅因ME、马萨诸塞MA、新罕布什尔NH、罗德岛RI、佛蒙特VT）+新泽西NJ、纽约东部、宾夕法尼亚东部、特拉华DE、华盛顿特区（部分） |
| 1          | 中大西洋：纽约州大部、宾夕法尼亚大部、宾州西部除外 |
| 2          | 东南部：弗吉尼亚VA、西弗吉尼亚WV、马里兰MD、北卡NC、南卡SC、乔治尼亚GA、佛罗里达北部 |
| 3          | 南部深区：阿拉巴马AL、田纳西TN、密西西比MS、路易斯安那LA、佛罗里达中南部 |
| 4          | 中西部+五大湖：肯塔基KY、印第安纳IN、俄亥俄OH、密歇根MI、伊利诺伊南部 |
| 5          | 中北部：威斯康星WI、明尼苏达MN、爱荷华IA、南达科他SD、北达科他ND、蒙大拿MT、伊利诺伊北部 |
| 6          | 南中部：得克萨斯TX、俄克拉荷马OK、阿肯色AR、密苏里MO、堪萨斯KS、科罗拉多东部 |
| 7          | 西南部：新墨西哥NM、亚利桑那AZ、犹他UT、内华达NV、加州南部 |
| 8          | 落基山脉+大平原西部：科罗拉多西部、怀俄明WY、爱达荷ID、蒙大拿西部 |
| 9          | 西海岸+太平洋+阿拉斯加、夏威夷：加州北部、俄勒冈OR、华盛顿州WA、阿拉斯加AK、夏威夷HI、关岛等 |

### 2. 前三位数字（SCF，Sectional Center Facility）：代表州内或跨州的主要邮政分拣中心
- 前三位通常对应一个大城市或主要邮政枢纽。
- 同一个州内，前三位数字基本连续或按地理位置顺序排列。
- 例子：
  - 纽约市：100xx（曼哈顿）、104xx（布朗克斯）、112xx（布鲁克林）等
  - 洛杉矶：900xx–908xx、910xx–918xx
  - 芝加哥：606xx–608xx
  - 休斯敦：770xx–778xx
  - 旧金山湾区：940xx–951xx

### 3. 前五位整体规律：从东向西、从北向南递增
- 东海岸：0xxxx–2xxxx
- 中部：3xxxx–6xxxx
- 西海岸：9xxxx
- 越往北，前缀数字往往越小；越往南越大（同一经度下）。
- 例子对比（大致同经度）：
  - 缅因州最北部：039xx
  - 佛罗里达最南部：33xxx–34xxx

### 4. ZIP+4（后4位）的意义
- 前5位确定到某个投递区域（通常一个小镇或城市的一部分）
- 后4位进一步细分：
  - 前2位：具体投递路线或邮局
  - 后2位：更细的段（街区、楼栋、甚至单侧街道）

### 5. 常见记忆口诀（中文网友总结）
0在东北角，123往下排，  
456中部转，789到西海岸。  
越往南数字大，越往西数字高。

### 6. 快速判断某个邮编大概在哪里（实用技巧）
- 0开头 → 肯定东北部（纽约、新泽西、马萨诸塞等）
- 9开头 → 肯定西海岸（加州、华盛顿州）或阿拉斯加/夏威夷
- 3开头 → 南部（佛罗里达、阿拉巴马、密西西比等）
- 75/76/77 → 几乎全是得克萨斯
- 60xxx → 伊利诺伊（芝加哥）或华盛顿州（西雅图）
- 100–119 → 纽约市（曼哈顿100xx，布鲁克林112xx等）

掌握了上面这些规律，你看到一个5位邮编基本就能立刻判断出它在美国的哪个大区、哪个州甚至哪个城市了。希望对你有帮助！
