# 亚马逊禁止商家干的事情 

亚马逊为维护平台公平竞争环境和消费者权益，制定了严格的卖家行为准则，禁止商家的违规行为涵盖评价操纵、运营作弊、侵权售假等多个维度，具体如下：
1.  **操纵评价与排名相关**
    - **违规干预商品评价**：禁止以返现、优惠券、赠品等报酬换取好评，也不能要求买家只留好评或修改、删除差评；同时严禁用虚假账号给自己商品刷好评，或给竞争对手恶意刷差评。比如包裹中夹带诱导好评的小卡片，就属于典型违规。
    -  **恶意操纵销售与搜索排名**：不允许通过虚假订单、自购商品等方式提升销量排名；也不能操纵关键词，或用机器人、付费刷点击等手段人为增加流量来抬高搜索排名；此外，订单确认后擅自提价、在商品描述中违规宣称销售排名等行为也被禁止。
2.  **商品与 listings 违规相关**
    -  **违规使用商品变体**：变体仅适用于尺寸、颜色等属性有差异的同类型商品。禁止将不相关商品捆绑作为变体销售，或通过关联高销量商品变体来提升低销量商品曝光等操纵行为。
    -  **商品信息与实际不符**：禁止夸大商品功能，比如普通日用品宣称有医疗功效；也不能隐瞒商品缺陷，如二手商品不说明磨损情况；同时不允许为同一商品创建多个详情页，或发布具有冒犯性的商品图片。
    -  **售卖禁限类商品**：严禁销售非法商品，如违禁药品、未经批准的处方药等；也不能卖假冒伪劣、盗版商品，以及违反濒危野生物种贸易公约等相关法规的商品；FBA卖家还需遵守危险品管理规定，未按要求处理的危险品商品无法通过FBA配送。
3.  **竞争与运营行为违规相关**
    -  **损害其他卖家权益**：禁止通过发布虚假订单占用竞争对手库存，或恶意举报竞品售假等方式，干扰其他卖家的正常运营。这类恶意竞争行为一旦查实，平台会严厉处罚。
    -  **绕过平台交易流程**：不允许用邮件、商品描述等方式添加外部链接、联系方式，诱导买家跳转至其他网站或私下交易；同时卖家只能通过亚马逊的买家 - 卖家消息服务联系买家，禁止私下打电话、加社交账号等违规沟通方式。
    -  **滥用平台功能与违规多账号运营**：禁止滥用促销、优惠券等工具获取不正当曝光；无合理业务需求时，不得经营多个卖家账户；也不能允许他人以违反亚马逊政策的方式代自己运营店铺。
4.  **其他违规行为**
    -  **侵犯知识产权**：未经授权不得使用他人商标、专利，也不能销售盗版书籍、影视、软件等侵犯著作权的商品。
    -  **违规使用买家信息**：卖家获取的买家手机号等信息仅可用于配送相关需求，严禁用于发送营销广告等无关用途，且需严格遵守买家个人信息保护规定。
    -  **绩效指标恶意规避**：自配送卖家不得故意突破迟发率、配送前取消率等绩效红线，若长期绩效不达标且无整改，会面临商品下架、账户受限等处罚。
